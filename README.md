# Skin Care Website 🌸

This is a simple, colorful, and interactive website designed to educate users about daily skin care.  
It covers topics like:

- Importance of skin care  
- Using sunscreen daily  
- Moisturizing the skin  
- A simple daily skin routine  

The site is built with **HTML, CSS, and JavaScript**, and includes a small interactive "Leave a Note" section for visitors to try.  
It is fully ready to be hosted on **GitHub Pages**.

[Visit the live site here](https://saraissa7.github.io/skin-care-website)
